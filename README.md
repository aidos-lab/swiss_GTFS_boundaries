# swiss_GTFS_boundaries
Repository generated to filter Swiss GTFS using different regional aggregation scales for further analysis. The aggregation scales available are Cantons, Communes, Distrcts and Aglomerations.

Swiss GTFS Data was obtained from https://opentransportdata.swiss/en/cookbook/timetable-cookbook/gtfs/


Aglomerations data was obtained from https://data.geo.admin.ch/browser/index.html#/collections/ch.bfs.generalisierte-grenzen_agglomerationen_g1
